What is the Bot for?

The Bot is for Simply run a Python Script from a Github Repo and you can also add packes from PyPi.org

Commands:

py3

with py3 you can run a python scipt from your GitHub Repo

py2

with py2 you can run a python scipt from your GitHub Repo

github

with github you can search through GitHub and find Repos

bug

If you found a bug please report it with this command ONLY USE THIS IN DMs

set

You can set the setting of your server[defaultrole, prefix, lvl] (CAN ONLY BE USED FROM ADMINS)

clear

You can clear channels with 'clear the number of deleted messages'

aliases

You can see all aliases for each command

stackoverflow

You can search for questions in Stack Overflow

lvl

You can see the xp or the lvl and the rank with 'lvl' or 'lvl @MEMBER'

leaderboard

You can see the leaderboard of the server(now bugged update comming soon)

for more informations you can type ("$help")
